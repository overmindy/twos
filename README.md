# 《两只(TWOS)》- 养成互动式情侣手札小程序

## 1. 项目愿景 (Vision)

《两只(TWOS)》不仅是一款情侣互动工具，更是一本“由两个人共同书写的私人手札”。它摒弃了现代社交应用的冰冷渐变，回归到纸质、笔触与墨痕的质感，通过仪式感的互动（如双盲写信、心跳共振、共养植物）来沉淀情感资产。

---

## 2. 视觉与体验灵魂 (Design DNA)

* **美学基调**：纸质书面（Paper & Literary）。采用暖米色道林纸纹理背景 (#FDFCF8)。
* **字体标准**：全面应用 **霞鹜文楷 (LXGW WenKai)**，提供人文书卷气息。
* **物理隐喻**：
  * **折叠便笺**：首页每日一问采用 3D 折叠动画，双方答题后“摊平”揭晓。
  * **朱砂印章**：所有操作入口（绑定、提交、设置）均设计为朱砂或墨黑印章。
  * **墨迹扩散**：共振与五子棋落子均带有 Canvas 实现的宣纸晕染效果。
  * **拍立得**：上传的图片自动包裹带有随机旋转角度的相纸边框。

---

## 3. 核心功能模块 (Modules)

### 3.1 每日一问 (The Daily Memo)

* **双盲解锁**：只有双方都提交回答后，才能“刮开”或“展开”查看对方内容。
* **多模态记录**：支持文字、语音（极简录制）、图片（裁切时光）。
* **墨迹隐去 (Privacy)**：用户可随时隐去自己的回答，对方查看时会显示模糊效果及“墨迹已隐”。
* **文笔润色**：集成 AI 语气实验室，将平实语言转化为具有美感的书面表达。

### 3.2 羁绊之树 (Botany Canvas)

* **白描生长**：首页背景为基于 Canvas 递归算法绘制的钢笔风格植物。
* **能量驱动**：互动产生的 `energy` 值决定树木的繁茂程度、分支深度及细节。
* **预见未来**：支持短时间加速预览树木长成后的终极形态。

### 3.3 冲突缓冲：止戈室 (The Peace Room)

* **独立聊天室**：专为产生分歧时设计的沟通空间。
* **止戈逻辑**：输入带有情绪的文字，AI 会强制拦截并润色为非暴力沟通的优美文句，给双方台阶。

### 3.4 状态气象站 (Weather Station)

* **实时同步**：基于云开发 Watcher，秒级同步双方的物理距离、实时天气及心情图标。
* **动态 TabBar**：底部“我”标签的图标会随当前心情动态变化。

### 3.5 手札小游戏 (Paper Games)

* **纸上抛硬币**：云端随机数共识，双方同步观看 3D 翻转动画。
* **墨色五子棋**：15x15 实时对弈，伴随落子墨迹与震动反馈。

## 4. 技术架构 (Technical Architecture)

* **前端**：原生微信小程序 (WXML/WXSS/JS)。
* **后端**：微信云开发 (Tencent CloudBase)。
  * **数据库镜像 (Info Mirroring)**：用户信息冗余存储在关系文档中，确保离线可见性。
  * **实时引擎**：利用 `db.watch` 实现跨设备、跨页面的毫秒级状态同步。
  * **4AM 机制**：业务日期判定标准为“当前时间减去4小时”，确保深夜互动不跨天。
* **多模态存储**：云存储路径规范化 `answers/{relId}/{timestamp}.{ext}`。

## 6. 环境部署指南

1. **云环境**：请确保 `miniprogram/envList.js` 中的 ID 正确。
2. **集合创建**：需手动或通过初始化脚本创建：`users`, `relationships`, `answers`, `daily_questions`, `games`, `gobang_games`。
3. **云函数上传**：每次修改 `cloudfunctions/quickstartFunctions/index.js` 后，**必须**右键点击文件夹选择“上传并部署：云端安装依赖”。
4. **图标路径**：自定义图标均存放于 `miniprogram/images/icons/`。
